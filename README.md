# rebalance-bot
Автоматический 50/50 BTC/USDT с порогом 4% бот ребалансирует при отклонении >4%, позволяя накапливать BTC при росте и USDT при падении ребаланс крипто-портфеля (BTC/USDT) с реал-тайм графиками (Chart.js), donut-диаграммой и Telegram-уведомлениями. Python backend + HTML/CSS/JS фронт. Поддерживает пресеты ETH/BTC/USDT и кастомные пропорции.
