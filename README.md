# Rebalance Bot — Автоматический ребаланс крипто-портфеля

**По сути этот бот "майнит" BTC и USDT без майнинга**  
Он не тратит электричество, не требует дорогого оборудования и не жрёт киловатты. Просто держит портфель в нужных пропорциях (например 50/50 BTC/USDT) и ребалансирует при отклонении >4%.  
При росте BTC → продаёт часть → покупает USDT (фиксирует прибыль в стейбле).  
При падении BTC → продаёт USDT → покупает больше BTC на дне.  
В итоге количество BTC постепенно растёт, а USDT накапливается как "буфер". Это своего рода "пассивный майнинг" через волатильность рынка — без ASIC, ферм и счетов за свет.

## Основные плюсы подхода
- Ноль затрат на железо и электричество  
- Работает 24/7 на любом VPS или домашнем ПК  
- Не нужно покупать BTC заранее большими суммами — можно начинать с любой суммы  
- Защищает от сильных просадок (больше USDT при падении)  
- Позволяет накапливать BTC "на автомате" при любом рынке  
- Простая настройка пресетов (включая ETH/BTC/USDT)  
- Полный контроль: графики, ROI, комиссии, уведомления в Telegram

## Как работает ребаланс (пример 50/50 BTC/USDT с порогом 4%)
Цель — 50% BTC и 50% USDT.  
Если доля BTC становится 54%+ или 46%- — бот автоматически продаёт/покупает, чтобы вернуть 50/50.  
Результат:  
- При росте цены BTC → фиксируешь часть в USDT → накапливаешь стейбл  
- При падении → докупаешь BTC дешево → увеличиваешь количество монет  
Со временем портфель становится всё "тяжелее" в BTC, но при этом остаётся сбалансированным и устойчивым.

Поддерживает любые пресеты, например:  
- BTC 30% / ETH 30% / USDT 40%  
- BTC 40% / ETH 20% / USDT 40%  
- Полностью кастомные пропорции

## Технологии
- Backend: Python (ccxt для Bybit, pandas)  
- Frontend: HTML5, CSS, JavaScript (vanilla)  
- Графики: Chart.js (линейные + donut)  
- Уведомления: Telegram Bot API  

## Как запустить локально
1. Склонируй репозиторий  
2. Установи зависимости: `pip install -r requirements.txt` (если есть)  
3. Запусти server.py  
4. Открой index.html через командную строку
5. Введи API-ключи Bybit и Telegram
